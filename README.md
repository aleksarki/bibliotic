# bibliotic
Web application for managing PDF files.


### frontend

deploy:
```
cd frontent
npm i
```

run:
```
cd frontend
npm start
```


### backend

deploy:
```
cd backend
npm i
```

run:
```
cd backend
npm run start
```


### pdf services

deploy:
```
cd pdf
pip install -r requirements.txt
```

run:
```
cd pdf
uvicorn app.main:app --reload
```
